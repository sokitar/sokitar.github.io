Word Comp
ZIP PASSWORD: wordcomp

¿Are you tired of comparing MS Word files one by one? Then this add-in is for you!
With this new functionality you will be able to compare the documents of a pair of folders of your choice.
USAGE
  1. Open Microsoft Word.
  2. Select Review -> Compare folders.
  3. A window will open were you will have to select 3 folders, writing the URL in the textbox or using the folder selector pressing the 'Pick' button:
    • Folder with originals: Pick the folder where your original files are located.
    • Folder with revisions: Pick the folder where the reviewed files are located.
    • Folder with comparisons: Pick the folder where you want to save the compared files.
  4. **OPTIONAL: Change the suffix that the add-in will use to find and generate your files.

    You can find an example below of how you should set the suffixes to obtain the best results.
  5. Click Compare to start the folder comparison
FORMAT EXAMPLE
This example shows how the suffixes work.

Supposing we choose _rev for the revised files and _comp for the compared files:
  1. If the originals folder contains:
    • original.docx
    • random.docx
  2. If the revisions folder contains:
    • original_rev.docx
    • original_2_rev.docx
    • smthng.docx
  3. After execution, the comparisons folder will contain:
    • original_comp.docx
    • original_2_comp.docx
